Seed Finder

This tool will help you determine which seed you are on for the purpose of using a step route.

Documentation

In order to use this tool, simply follow these steps after resetting at Mist:

  1. Every time you get an encounter, select the tile on which you got the encounter. Optionally, select the exact formation you got by clicking on it. (This is highly recommended, as it will help distinguish seeds, and is essential in case of certain twin seeds.)
  2. When you're done with a given map, simply click Next Image to move to the next map.
  3. Once you reach the save room in Watery Pass-South B2F, click the Finish button. This will lock in your encounters and help distinguish between seeds that only differ in whether or not they get one last encounter.

There are several pairs of seeds that can't be distinguished by steps alone by the time you reach the save room. We call these twin seeds. Some of these can be distinguished by formation, which is why it is highly recommended that you select formations. However, some of them cannot be distinguished until after the save room. If you reach the save room and are left with twin seeds, you should open both routes and try to figure out which one is the correct one.

The current beta step routes are not twin seed safe. One of the twins may require taking steps in the save room and the other may not. In this case, you will have to guess and hope you're right. This will eventually be corrected.

Roadmap

The following features are planned, and may or may not actually happen depending on if I feel like working on this:

  • Add additional maps after the save room in order to help with twin seeds. This is more complicated because the required data is route-dependent and can't be done until the step routes are finalized and twin seeds are corrected.
  • Allow for fuzzy searches. In particular, I'm thinking of two instances: First, the tool currently requires that you do not skip any encounters. If you do, it will return incorrect results. I'd like to make it possible to optionally be more lenient (though it may end up impossible to identify your seed in that case). Second, I'd like to add a feature to help in cases where you weren't sure exactly which tile you got an encounter on.
